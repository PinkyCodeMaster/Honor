## Install

```sh
pnpm install
```

## Usage

```sh
# Start development server
pnpm run dev
```

```sh
# Update all shadcn/ui components to their latest versions
# This will overwrite existing components with the latest changes
pnpm run ui:update
```

## Run tests

```sh
pnpm run test
```
